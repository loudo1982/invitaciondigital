# Invitación Digital – Starter

Este paquete incluye los archivos a **agregar o reemplazar** en tu proyecto Next.js recién creado.

## Pasos rápidos
1) Instala Tailwind y dependencias:
   npm i tailwindcss postcss autoprefixer

2) Copia el contenido de este zip en la raíz de tu proyecto (respetando carpetas).

3) Agrega una imagen a public/imgs/sample-cover.jpg (ya hay un placeholder).

4) Ejecuta:
   npm run dev
   y visita /invitacion/nicole-renata

Ajusta lib/invitations.js con tus datos reales (WhatsApp, direcciones, etc.).
